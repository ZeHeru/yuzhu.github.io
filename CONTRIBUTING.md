# Contributing

This is a personal academic website. Most changes should be made directly to content files and reviewed with a local Jekyll build before publishing.

Useful entry points:

- `_pages/about.md` for the homepage.
- `_pages/cv.md` and `files/cv.pdf` for CV updates.
- `_publications/` for publication entries.
- `_portfolio/` for project entries.
- `_config.yml` and `_data/authors.yml` for profile metadata.
